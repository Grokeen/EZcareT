# 카드 취소 방법
> 주의 : 묶음 수납한 걸 취소하려면 안된다고 한다(2024-09-11)

## 1. 외래진료비수납/예약 화면
- 이미 수납이 Y인 건 선택
- 계산 버튼 클릭 
- 환불 버튼 활용

> 주의 : 키오스크 결제 시, 오류는 즉각 보고

## 2. 외래진료비수납상세내역 화면
- 외래진료비수납/예약 화면 하단에 외래진료비수납상세내역 버튼 클릭
- 환자번호와 날짜 조회
- 원하는 건 선택 후 취소

## 3. 과거일자카드취소 화면

> 참고 : 카드내역조회 화면도 있으니 참고
